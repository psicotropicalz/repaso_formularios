Reservas

Reservar pistas

Reserve una pista o cancha para su actividad deportiva favorita. Ofrecemos una variedad de canchas para diferentes deportes como tenis, baloncesto, y fútbol. Asegúrese de reservar con antelación para garantizar disponibilidad.

Reservar pabellón

Reserve el pabellón para eventos o actividades deportivas. Nuestro pabellón es ideal para grandes eventos, torneos y actividades grupales. Equipado con todas las facilidades necesarias para asegurar el éxito de su evento.
